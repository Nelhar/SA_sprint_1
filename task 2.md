## Список систем
**ИС «Алмазы»** (существующая) - отбор камней для выставления лота
**ИС «Аукцион»** (новая)- проведение онлайн аукциона (требуется создать площадку для проведения аукциона)
**ИС «Логистика»** (существующая) - перевозка алмазов  
**ИС «Бухгалтерия»** - оплата услуг внешнего сервиса (новая и требует доработки), ведение учета аукциона (существующее, требует доработки для полноценной цифровизации), оплата логистики (существующее)
**ИС «Лаборатория»** (новая система) - оцифровка и подготовка полноценной карточки камня. 
**Частное облако (внешний сервис)** (новый) - сервис для временного хранения данных о камнях (временное решение) 


## Список интеграций
**ИС «Алмазы» → ИС «Логистика»**: планирование и перевозка алмазов. (существующая).
**ИС «Лаборатория» → ИС «Аукцион»**: передача описания и 3D моделей алмазов (новая). для MVP - ручная отправка/загрузка в облачный сервис. Для Целевой - интеграция REST
**ИС «Лаборатория» → ИС «Бухгалтерия»**: актирование и оплата услуг лаборатории (новая). для MVP - ручная отправка/загрузка в облачный сервис. Для Целевой - интеграция REST
**ИС «Логистика» → ИС «Бухгалтерия»**: актирование и оплата услуг транспортной компании (существующее).
**ИС «Аукцион» <-> ИС «Бухгалтерия»**: регистрация оплаты выкупленного лота (существующая).
